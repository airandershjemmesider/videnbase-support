# Guide til valg af hosting

## Problem

Du skal vælge den rigtige hosting-type til en hjemmeside — men der er mange muligheder med meget forskellig pris og ydeevne.

## Hosting-typer

### Shared hosting (delt hosting)
- **Hvad:** Mange hjemmesider deler én server
- **Pris:** Ca. 20-100 kr/md
- **Fordele:** Billigt, nemt at komme i gang, hosting-firmaet håndterer alt
- **Ulemper:** Langsom ved spidsbelastning, begrænset kontrol, naboeffekt (andre sites påvirker din)
- **Passer til:** Små hjemmesider, blogs, foreningssider, enkle virksomhedssider
- **Eksempler:** Simply.com, one.com, DanDomain, GoDaddy

### Managed WordPress hosting
- **Hvad:** Hosting optimeret specifikt til WordPress
- **Pris:** Ca. 100-500 kr/md
- **Fordele:** Hurtig, automatiske opdateringer, staging, WordPress-support, daglige backups
- **Ulemper:** Kun WordPress, dyrere end shared
- **Passer til:** Professionelle WordPress-sider, virksomheder med WordPress-fokus
- **Eksempler:** Kinsta, WP Engine, Cloudways, SiteGround

### VPS (Virtual Private Server)
- **Hvad:** Din egen virtuelle server med dedikerede ressourcer
- **Pris:** Ca. 50-500 kr/md
- **Fordele:** Fuld kontrol, dedikerede ressourcer, skalerbar
- **Ulemper:** Kræver teknisk viden, du styrer selv sikkerhed og opdateringer
- **Passer til:** Flere sites, webshops, sites med højt trafik, udviklere
- **Eksempler:** DigitalOcean, Hetzner, Linode, Vultr

### Dedikeret server
- **Hvad:** En hel fysisk server kun til dig
- **Pris:** Ca. 500-5.000+ kr/md
- **Fordele:** Maksimal ydeevne og kontrol
- **Ulemper:** Dyrt, kræver serveradministration
- **Passer til:** Store webshops, højtrafiksider, specielle krav

## Løsning: Vælg baseret på behov

### Spørgsmål at stille

| Spørgsmål | Anbefaling |
|-----------|------------|
| Simpel side med lidt trafik? | Shared hosting |
| WordPress-side der skal være hurtig? | Managed WP hosting |
| Flere sider eller teknisk kontrol? | VPS |
| Webshop med mange besøgende? | Managed WP eller VPS |
| Budget under 50 kr/md? | Shared hosting |

### Vigtige ting at tjekke

1. **Server-placering:** Vælg en server i Danmark/Nordeuropa for danske sider
2. **PHP-version:** Mindst PHP 8.0+ (gerne 8.2+)
3. **SSL inkluderet:** De fleste inkluderer Let's Encrypt gratis
4. **Backup:** Automatiske daglige backups (og nem gendannelse)
5. **Oppetid (uptime):** Minimum 99,9% garanti
6. **Support:** Dansk support er en fordel for danske kunder
7. **Skalerbarhed:** Kan du opgradere uden at flytte?

## Bemærkninger

- **Undgå ukendte billig-hosts** — dårlig support og sikkerhed koster mere i længden
- **Shared hosting er fint til de fleste** — du behøver ikke VPS til en simpel virksomhedsside
- **Managed WP hosting er det bedste kompromis** for professionelle WordPress-sider
- Start billigt og opgrader når behovet opstår — de fleste hosts gør det nemt at skifte plan
- Tjek anmeldelser på Trustpilot og hosting-fora inden du vælger
